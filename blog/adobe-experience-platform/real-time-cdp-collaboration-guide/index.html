Real-Time CDP Collaboration: Complete Guide to Brand-to-Brand Data Partnerships

Adobe Real-Time CDP Collaboration is transforming how brands partner on audience data. With global expansion, multiple match keys, and brand-to-brand collaboration now available, here's everything you need to know about this powerful capability.

Now Globally Available: As of December 2025, Real-Time CDP Collaboration is available in North America (including Canada), Australia, New Zealand, and EMEA. It's automatically available to Real-Time CDP Prime and Ultimate customers.

What is Real-Time CDP Collaboration?

Real-Time CDP Collaboration is Adobe's solution for secure, privacy-compliant data partnerships between brands. Think of it as a data clean room built directly into Adobe Experience Platform that enables:

  • Audience sharing between partners without exposing raw customer data
  • Overlap analysis to understand shared audience segments
  • Joint activation to reach customers across partner destinations
  • Measurement of campaign performance across partner data

Whether you're an advertiser looking to reach publisher audiences, or two brands wanting to leverage each other's customer insights, Collaboration provides the infrastructure to do it securely.

Supported Match Keys

Match keys are the identifiers used to find overlapping audiences between collaborators. Real-Time CDP Collaboration now supports six match key types:

Match Key Description Best For
Hashed Email SHA-256 hashed email addresses Highest match rates; universal identifier
Hashed Phone SHA-256 hashed phone numbers Mobile-first audiences; SMS marketing
CRM ID Your internal customer identifier B2B partnerships; known customer matching
Loyalty ID Loyalty program identifiers Retail partnerships; coalition loyalty programs
Hashed IPv4 SHA-256 hashed IP addresses Household-level targeting; connected TV
AdFixus ID AdFixus identity graph ID Cookieless identity resolution

Using Multiple Match Keys

One of the most powerful recent updates is support for multiple match keys simultaneously. This dramatically expands your potential audience size and improves match rates.

Multiple match keys can be used when:

  • Sourcing audiences — Pull in audiences using multiple identifiers
  • Establishing connections — Match with collaborators across multiple ID types
  • Activating audiences — Send audiences with multiple keys to destinations
Important: When activating audiences with multiple match keys, if any match key has no overlaps, no audience counts, or falls below the minimum threshold, the entire activation will fail. Ensure your audiences have sufficient overlap and meet the minimum threshold of 1,000 IDs across all match keys before activating.

Brand-to-Brand Collaboration

The July 2025 release introduced a game-changing capability: brand-to-brand collaboration. Previously, Collaboration was limited to advertiser-publisher relationships. Now, any two brands can partner.

Flexible Partnerships

No longer limited to advertiser-publisher model. Two retailers, two financial services companies, or any brand combination can collaborate.

Private Connections

Share your unique connect code with partners who can then connect directly—no need to be in the public Discover directory.

Bidirectional Activation

Both collaborators can activate audiences, not just one direction. Configure activation permissions when establishing connections.

Audience Index Scores

Measure how well your audience matches your collaborator's with index scores based on underlying audience counts and overlaps.

Collaboration Patterns

Real-Time CDP Collaboration supports two collaboration patterns:

  1. Advertiser-to-Publisher: The traditional model where advertisers collaborate with publishers to reach their audiences and measure campaign performance.
  2. Brand-to-Brand: Two brands (regardless of advertiser/publisher role) collaborate to share audience insights, expand reach, and jointly activate audiences.

Self-Serve Destinations

The My destinations tab in the Setup section now allows both advertisers and publishers to configure destinations self-serve. This means you can:

  • Activate audiences to third-party platforms (ad networks, DMPs, etc.)
  • Configure Adobe Experience Platform as a destination
  • Use linked keys to specify which match key is sent during activation
  • View Collaboration audiences in the AEP Audience Portal
Note: Currently, only Adobe Experience Platform destinations are available for self-serve configuration. For other destination types, contact your Adobe representative.

Managing Audiences at Scale

The August 2025 release introduced a bulk edit tool that makes managing multiple audiences significantly easier:

  • Edit audience metadata for multiple audiences at once
  • Update connection access permissions in bulk
  • Modify names, descriptions, and categories across audiences
  • Configure audience refresh frequency (daily or every 2-6 days)

Best Practices for CDP Collaboration

1. Start with High-Quality Match Keys

Your match rates are only as good as your data quality. Before starting a collaboration:

  • Ensure email addresses are properly formatted and hashed consistently (SHA-256)
  • Standardize phone numbers to E.164 format before hashing
  • Validate CRM and Loyalty IDs are populated across your customer base

2. Validate Overlap Before Activation

Use the Discover tab to analyze audience overlaps before activating. The new audience index scores help you understand:

  • How well your audience matches your collaborator's
  • Which segments have the highest overlap potential
  • Whether you'll meet the minimum 1,000 ID threshold

3. Leverage Multiple Match Keys Strategically

Don't just enable all match keys—be strategic:

  • Start with your strongest identifier (usually hashed email)
  • Add secondary keys to expand reach incrementally
  • Monitor match rates and adjust key combinations

4. Configure Refresh Frequency Appropriately

Choose your audience refresh frequency based on campaign needs:

  • Daily refresh: For real-time campaigns, flash sales, or rapidly changing audiences
  • 2-6 day refresh: For evergreen campaigns or stable audience segments

Getting Started

Ready to start using Real-Time CDP Collaboration? Here's your action plan:

  1. Verify eligibility: Confirm you have Real-Time CDP Prime or Ultimate
  2. Set up match keys: Configure which identifiers you'll use for matching
  3. Source your audiences: Import audiences into Collaboration
  4. Find collaborators: Use Discover directory or private connection codes
  5. Establish connections: Configure activation permissions and access
  6. Analyze overlaps: Use Discover tab and index scores
  7. Activate audiences: Send to destinations or receive from collaborators
